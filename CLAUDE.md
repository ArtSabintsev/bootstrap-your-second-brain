# Agent bootstrap (CLAUDE.md / GROK.md / CODEX.md)

This repo is a config-driven second brain: an Obsidian vault plus the automation
that feeds it, and the persistent memory agents load for context. The schema,
rules, and conventions live in `AGENTS.md`, the single source of truth. Load and
follow it.

**Session start (non-trivial work):** read `Profile/00-overview.md` and
`Profile/working-with-me.md` before answering.

**Standing write phrases:** "log this", "remember this", "kill this idea"
append to `Log/YYYY-MM-DD.md` only (see Operator log in `AGENTS.md`).

**If asked to bootstrap / set up this vault:** follow `SETUP.md` end to end
(you may write `config.json`, seed Profile, place secrets under secrets_dir,
run bootstrap + doctor). Do not push unless the operator says to.

`GROK.md` and `CODEX.md` are symlinks to this file so every harness entry point
stays identical by construction. Edit this file only.
